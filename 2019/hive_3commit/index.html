Hive命令三种提交方式

Written by Darren.Chen with ♥ on March 11, 2019 in Hive

命令行界面

命令行界面是和hive交互最常用的方式 进入方式：${HIVE_HOME}bin/hive

例如
hive> select * from test2;
OK
2
3
Time taken: 1.549 seconds, Fetched: 2 row(s)
命令行一些常用的功能
  1. 自动补全功能：如果用户在输入的过程敲击Tab键，那么hive会帮你自动补全关键字或者函数，方便我们开发和减少我们开发时间。
  2. 显示当前数据库名字：在conf目录下的hive-site.xml加入 hive.cli.print.current.db true
  3. 显示字段名字 hive.cli.print.header true 提示:如果conf下没有hive-site.xml，则需要我们自己创建 <?xml-stylesheet type=“text/xsl” href=“configuration.xsl”?> hive.cli.print.header true hive.cli.print.current.db true
  4. 在hive使用hadoop命令 用户可以在hive使用hadoop命令，这样会更加高效写，因为在hive会在同一个线程执行这些命令，而在 bash shell需要每次执行都启动一个新的线程。 例如：hive (default)> dfs -ls /;

在bash shell上使用hive

使用 -e 不进入hive的交互窗口执行sql语句
    bin/hive -e "select * from test2;"
使用-f 执行脚本中sql语句 文件的后缀为.q或者.sql
touch test2.sql
vim test2.sql
插入hql语句
select * from test2;
insert into test2 values(7);
select * from test2;

bin/hive -f /opt/test2.txt;

额外 加入-S 可以将结果保存在一个文件中

    例如 bin/hive -f /opt/test2.txt > /opt/result.txt;
